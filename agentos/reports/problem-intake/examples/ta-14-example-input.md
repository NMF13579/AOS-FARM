# Non-production test input: Local Notes Organizer

## Problem Statement
The user struggles to keep track of daily thoughts and tasks without getting overwhelmed by complex features in modern note-taking apps.

## Target User
A non-programmer user who wants a simple, distraction-free environment for taking notes.

## Current Workflow
Currently writing notes on physical sticky notes that get lost or taking notes in a default OS text editor that lacks organizational features.

## Desired Outcome
A small local notes organizer that allows creating, editing, and deleting plain text notes with a minimal UI.

## Constraints
Must run locally. No cloud sync required. No complex markdown rendering needed.

## Known Risks
May lack backup if the user's local disk fails. The user might eventually want sync features that are explicitly out of scope.

## Open Questions
Should the notes be searchable? Should we support basic tags?
