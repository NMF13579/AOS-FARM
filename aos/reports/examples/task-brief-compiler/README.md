# Task Brief Compiler Examples

These examples show the review-only output shape for a Task Brief draft that
comes from an accepted candidate plus a valid selection decision.

They are not executable tasks, not execution authorization, and not approval.

## Included Example

- `task-brief-draft-example.md`

## Safety Notes

- The compiler may create a Task Brief draft only.
- The draft is not executable until human review and explicit authorization.
- The draft does not assign Risk Profile.
- The draft does not authorize execution.
- The draft does not authorize commit or push.
- The draft does not start the next task.
- PASS is validation result only, not approval.
- Evidence is not approval.
- CI PASS is not approval.
- UNKNOWN is not OK.
- NOT_RUN is not PASS.
- Human approval cannot be simulated.
