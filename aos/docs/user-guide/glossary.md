# AOS Glossary

- **Task Brief**: A clear definition of the work to be performed, constraints, and success criteria.
- **Execution Report**: An artifact generated by the agent detailing what was changed and verifying boundaries were kept.
- **Human Checkpoint**: An explicit decision point where a human authorizes a task, assigns a Risk Profile, or approves a commit.
- **Risk Profile**: A classification of a task's potential impact (e.g., `LOW_RISK_FAST`, `HIGH_RISK_PROTECTED`).
- **Baseline**: The clean, verified state of the git repository branch before a task begins.
