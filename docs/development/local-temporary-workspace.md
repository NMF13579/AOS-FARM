# Local Temporary Workspace Policy

This policy governs the usage of the local temporary workspace `.aos-tmp/` within the AOS-FARM repository and target consumer projects.

## Core Rules

1. **Location**: The temporary workspace must be located at `/.aos-tmp/` relative to the repository root.
2. **Ignored by Git**: The `.aos-tmp/` directory must be added to `.gitignore`.
3. **Usage**: All temporary command outputs, interim logs, scratchpad items, and intermediate test states generated by agents must be routed here.
4. **Local-Only**: This directory is local-only and disposable. It can be safely deleted or ignored without impacting project integrity.
5. **Not a Source of Truth**: `.aos-tmp/` is strictly ephemeral. It must **never** be used as a Source of Truth.
6. **No Evidence/Approvals**: You must **never** store Evidence, final reports, checkpoints, or human approval documentation in `.aos-tmp/`. These artifacts belong in tracked directories (e.g., `reports/`).

Agents are instructed to generate scratch files in `.aos-tmp/` instead of cluttering the root directory.
